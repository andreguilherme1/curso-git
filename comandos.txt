git init - Criar um novo repositório
git clone + URL - Clonar um repositório
git add . - Adicionar todos os arquivos
git commit -a -m "msg" - Commitar todas as mudanças
git push - Enviar os arquivos commitados
git pull - Recebe as alterações feitas
git status - Mostra o status
git branch - Listar as branches
git checkout (branch) ou (tag) - Para alternar entre branches ou tags
git checkout -b (nomedabranch) - Cria uma nova branch e alterna para ela
git checkout (nomedoarquivo) - Desfaz as mudanças em um arquivo
git log - Mostra as alterações feitas desde o início
git fetch - Atualiza de todos os branchs e tags não reconhecidos
git merge + branch - Junta a branch atual com a outra escolhida
git remote rm origin - Remover url de repositório
git submodule add (repo) - Adicionar o submódulo
git submodule - Verificar os submódulos
